# Selenium Test Automation Framework

This project is a web automation testing framework developed to automate UI testing for web applications.

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Allure Report

## Framework Features

- Page Object Model (POM) design pattern
- Organized test structure
- Automated test execution using TestNG
- Detailed test reporting using Allure Report
- Easy project build and dependency management with Maven

## How to Run the Tests

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the master_suite.xml file to execute the tests
